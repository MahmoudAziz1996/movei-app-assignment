## Installation

```sh
  git clone https://github.com/MahmoudAziz1996/movies-app-assignment
  
  npm install or yarn install
  
  npm start
```
## Run on Android

```
react-native run-android
```

## Run on iOS

```
 cd ios & pod install
 cd ..
 react-native run-ios
```
